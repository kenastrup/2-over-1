= Responsiv dobling

Hvis de melder og støtter hverandre og makker har vært inne i bildet med egen farge eller dobling. Viser kort og de to umeldte fargene

(1 [red]#♥#) - X - (2 [black]#♠#) - X

(1 [red]#♥#) - 1 [black]#♠# - 2 [red]#♥# - X

Hva med

(1 [red]#♦#) - X - 1 [red]#♥# - X

Responsiv den også!
