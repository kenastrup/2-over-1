== 1 [.hearts]#♥# -> 1 [.spades]#♠#
Naturlig. Dette viser 6+ hp og 4+ farge. Kan være råsterk også.

NOTE: Hvis man har GF kort og en annen 4/5 kortsfarge melder vi den andre først på to trinnet for å etablere GF, og melder sparen senere hvis behov. Siden vi melder veldig naturlig er det ikke noe stress med å vise fram sparen først. Åpner vil melde spar med 4-kort for å beskrive "formen" på hånda.

== 1 [.hearts]#♥# -> 2 [.spades]#♠#

Sperr 0-5 poeng, 6+ kort. OK farge.

== Støtt med støtte

Hvis vi hør støtte til makkers major-åpning bruker vi disse meldingene.

=== Nøyaktig 3 kort støtte

|===
| styrke | melding

| 5-6
| 1M -> 1 NT -> 2 X -> 2M. +
  Må gå via 1 NT med "søppel". +
  NB! Denne sekvensen kan også vise to kort farge-preferanse.

| 7-9
| 1M -> 2M. +
  Direkte støtte viser sterkere enn via 1 NT

| 10-12
| 1 M -> 1NT -> 2x. -> 3 M +
  Hoppstøtte via 1 NT. Viser limit raise


| GF
| Meld først 2/1, deretter støtte makkers major. Dette viser ALLTID 3 korts støtte.
|===
